1)  File : modeInfo.json
    Path : "C:\Users\Kripa.Shankar\Desktop\Mugen\configurations"
    Use : this file is used to check the access of artist,albums,songs available to different users
          We provide 2 features using this file - a. usermode, b. friendmode
            a. usermode : when user is logged in, he/she is in usermode, i.e he/she has access to artist,albums,songs
               available only to him/her.
            a. friendmode : user can log-in to this mode in his/her profile, In this mode he/she has access to artist,albums,songs
               available to him/her + access to artist,albums,songs of his friends(view only)
